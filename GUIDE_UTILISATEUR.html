1. Vue d'ensemble

YouTube Downloader Node est une application web locale qui tourne sur votre PC. Elle offre un ensemble complet d'outils pour telecharger, streamer et organiser du contenu YouTube.

⬇

Telecharger

Videos et audios en MP3, FLAC, WAV, AAC, OGG, MP4, MKV, WEBM avec choix de qualite

▶

Streamer (Mon Flow)

🔍

Playlists

2. Installation et lancement

Lancement rapide

Double-cliquer sur start.bat. Le script fait tout automatiquement :

  • 1
    Verification Node.js — verifie que Node.js est installe sur le systeme
  • 2
    Detection des outils — detecte automatiquement Python, ffmpeg et yt-dlp
  • 3
    Installation auto — installe les dependances manquantes via winget ou pip
  • 4
    Serveur — lance le serveur Express sur http://localhost:3000
  • 5
    Navigateur — ouvre automatiquement la page dans votre navigateur
💻 Capture : Terminal de lancement Lancer start.bat et capturer le terminal avec le message "Server running on port 3000".

3. Onglet Telecharger

C'est l'onglet principal. Il permet de telecharger une video ou un audio depuis une URL YouTube.

Etape 1 : Coller l'URL

Collez une URL YouTube dans le champ de saisie :

https://www.youtube.com/watch?v=dQw4w9WgXcQ
URLs acceptees :
  • https://www.youtube.com/watch?v=...
  • https://youtu.be/...
  • https://www.youtube.com/shorts/...
  • URLs de playlists YouTube
📷 Capture : Champ URL Coller une URL YouTube dans le champ et capturer avant de lancer.

Cliquez sur "Telecharger". L'application :

  1. Recupere les infos de la video (titre, miniature, chaine, vues)
  2. Affiche une carte video avec ces informations
  3. Lance le telechargement en arriere-plan via un worker
  4. Affiche une barre de progression en temps reel

6. Historique des telechargements

Accessible depuis la Bibliotheque via le bouton Historique

L'historique conserve les 200 derniers telechargements avec titre, URL, format, chaine, miniature et date.

Filtres

  • Recherche textuelle par titre
  • Tri : Date, Titre, Type
  • Presence :
Tous Presents Absents
  • "Presents" : fichiers encore dans la bibliotheque
  • "Absents" : fichiers supprimes (possibilite de re-telecharger)

Re-telechargement en lot

  1. Cocher les elements souhaites (ou "Tout cocher")
  2. Choisir un format (MP3, MP4, ou "Format d'origine")
  3. Cliquer sur Re-telecharger la selection
  4. L'application telecharge avec un delai de 3s entre chaque (protection anti-rate-limit)

Import / Export

Exporter la liste Importer une liste

Sauvegardez votre historique en JSON ou importez un historique precedemment exporte.

10. Extension Chrome

L'extension Chrome permet d'interagir avec l'application directement depuis les pages YouTube.

Installation

  • 1
    Ouvrir Chrome et aller dans chrome://extensions/
  • 2
    Activer le Mode developpeur (interrupteur en haut a droite)
  • 3
    Cliquer sur "Charger l'extension non empaquetee"
  • 4
    Selectionner le dossier extension_v2/ du projet
