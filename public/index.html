📖 Guide d'utilisation

Toutes les fonctionnalites expliquees + mini-tutos pas-a-pas pour bien demarrer.

Vue d'ensemble Telecharger Recherche Bibliotheque Mon Flow Paroles Karaoke Stats Raccourcis Depannage
🗺 Vue d'ensemble des onglets
  • Telecharger : colle une URL YouTube et choisis le format (audio MP3/FLAC/WAV/AAC/OGG ou video MP4/MKV/WEBM). File d'attente integree.
  • Recherche : recherche YouTube directement. 4 actions par resultat (DL, file, ajout Flow, ecouter en streaming).
  • Bibliotheque : tes fichiers locaux + sous-section Historique avec re-telechargement par lot.
  • Mon Flow : ton "Spotify perso". Streaming sans telechargement, playlists, top ecoutes, vues rapides (Aujourd'hui/Semaine/Mois...).
  • Stats : statistiques d'ecoute, ecoutes ephemeres, comparatif des strategies de stockage.
  • Profil : preferences par defaut, spectre visuel, options crossfade, notifications, paroles.
  • Aide : tu y es 👋
⬇ Telecharger un morceau (tutoriel)
  1. Onglet Telecharger
  2. Colle l'URL YouTube dans le champ
  3. Clique sur la barre "MP3 - Meilleure qualite" pour deplier les options : choisis Audio/Video, format, qualite, dossier de destination, telechargement de la couverture
  4. Clique Telecharger
  5. La barre de progression s'anime. Tu peux annuler a tout moment
  6. Quand c'est fini, lien direct vers le fichier (et la couverture si demandee)

Astuce : le bouton "+ Ajouter a la file d'attente" permet d'enchainer plusieurs URLs. Erreur ou message bloque ? Bouton × sur chaque message ou "Effacer le panneau".

🔍 Recherche YouTube + ecoutes ephemeres

Faire une recherche

  1. Onglet Recherche, tape un mot-cle, Enter ou bouton "Rechercher"
  2. Pour chaque resultat : DL telecharge directement / + File ajoute a la file / + Flow ajoute a Mon Flow / 🎧 Ecouter streame sans telecharger / ▶ Video ouvre la video
  3. Filtres en haut des resultats : Tous / Nouveaux / Telecharges / Mon Flow + filtre par duree
  4. "Charger plus de resultats" en bas pour passer a 40, 60... (max 50 par requete)

Recherches recentes

Quand le champ est vide, tu vois tes recherches recentes (10/20/50/100 affichables, configurable dans Profil). Clique pour relancer, croix pour supprimer.

Ecoutes ephemeres

Les titres ecoutes via 🎧 Ecouter mais jamais ajoutes a Flow/Lib apparaissent dans le bloc "Ecoutes ephemeres" sous les recherches recentes. Boutons : ▶ re-ecouter, + Flow, DL, × oublier.

📚 Bibliotheque + Historique

Bibliotheque

  • Statistiques en haut : total, audio, video, espace disque, duree totale
  • + Nouveau dossier pour organiser ; deplacement par menu sur chaque fichier
  • Tri (date / titre / type / taille), filtres par type/format, recherche textuelle
  • Selection multiple : Tout selectionner -> Lire la selection ou Supprimer la selection

Historique (bouton "Historique" en haut)

  • Filtres : Tous / Presents (encore dans la lib) / Absents
  • Re-telechargement par lot avec choix du format
  • Export JSON pour sauvegarder ton historique, Import JSON pour le recharger sur un autre PC (dedoublonnage automatique par URL)
  • Drag & drop d'un element vers Mon Flow
🎵 Mon Flow - ton Spotify perso

Ajouter des morceaux

  • + Depuis l'historique : importe tous tes telechargements passes
  • Rechercher sur YouTube : recherche integree, ajout direct
  • + Ajouter par URL : coller une URL YouTube
  • Drag & drop depuis l'historique vers Mon Flow ou une playlist
  • Extension Chrome : bouton +F sur YouTube
  • Importer : charger un JSON

Couverture Top N en haut

Les 3, 5, 10 ou 20 morceaux les plus ecoutes (selecteur dans le coin) sont mis en valeur en grandes cartes avec rangs or/argent/bronze. Ils restent aussi dans la liste en dessous.

Vues rapides (chips)

  • ⭐ Plus ecoutes · 🕐 Recemment ajoutes · ❤ Aimes
  • ☀ Aujourd'hui · 📅 Cette semaine · 🗓 Ce mois (basees sur lastPlayed)
  • 📷 Decouvertes : ajoutes <30j et deja ecoutes 3x+
  • 🔗 A redecouvrir : ecoutes 5x+ mais pas joues depuis >90j
  • 🕑 Coups d'un soir : joues 1 seule fois il y a >90j
  • ❓ Jamais ecoutes

Lecture (deux boutons par titre)

  • ▶ Ecouter : lecture sequentielle (desactive l'aleatoire)
  • 🔀 Aleatoire : lit ce titre + active le shuffle pour la suite
  • ▶ Video : ouvre l'iframe YouTube
  • ♥ : aimer/desaimer
  • × : retirer du Flow (envoye dans la corbeille 24h)

Crossfade (configurable dans Profil)

Slider 0-8s. A activer dans Profil → Lecture - Crossfade. Fondu enchaine entre titres consecutifs (audio uniquement). 0 = desactive.

Player bar global

3 modes mutuellement exclusifs : 🔀 Aleatoire, 🔁 Repeter selection, 🔂 Repeter ce titre. Quand un mode est actif il est visible (fond rouge plein + petit point lumineux).

📜 Paroles - 3 sources, recherche manuelle, versions

Comment ouvrir

Lance un titre, le player bar apparait en bas. Clique 📜 a droite (entre Sleep timer et File de lecture). Le panneau s'ouvre en bas a droite.

Sources interrogees automatiquement

  1. lrclib.net (paroles synchronisees LRC, le mieux)
  2. api.lyrics.ovh (paroles plain en fallback)
  3. genius.com (scraping en dernier recours, le plus complet)

Spinner anime pendant la recherche. A 1.5s : "Premiere source vide, exploration des autres sources..." s'affiche.

Si les paroles sont fausses ou le matching rate

Bouton ✏️ dans la barre d'outils du panneau ouvre un mini-formulaire Artiste / Titre pre-rempli. Corrige et clique Chercher. Chaque recherche reussie devient un onglet en haut du panneau (jusqu'a 5) - clique pour comparer / switcher.

Si rien n'est trouve

7 liens cliquables vers Genius, AZLyrics, Musixmatch, Paroles.net, Lyrics.com, Google « lyrics », YouTube « lyrics » apparaissent automatiquement.

Reglages dans la barre d'outils du panneau

  • ☑ Suivi auto : on/off du scroll automatique
  • Mode click sur une ligne : Seeker (la chanson saute a ce moment), Synchro (cette ligne devient "maintenant", offset auto-calcule), Rien
  • Badge offset : si decalage applique, clic dessus = reset
  • 🎯 Recentrer sur la ligne courante
🎤 Mode karaoke plein ecran

Bouton 🎤 dans la barre d'outils du panneau Paroles. Defilement fluide type prompteur (transition 0.55s, mask gradient en haut/bas).

Reglages personnalisables

Icone ⚙️ en haut a gauche pendant le karaoke ouvre le panneau de reglages :

  • Effet ligne courante : Glow / Pulse / Bounce / Zoom max / Slide / Statique
  • Couleur : Rose-orange / Bleu / Vert / Violet / Or / Blanc
  • Lignes voisines : 1 / 2 / 3 avant et apres
  • Taille : Petit / Moyen / Grand / Tres grand
  • Police : Sans-serif / Serif / Monospace / Display (Impact)
  • Background : Sombre uni / Degrade sombre / Aurora animee / Noir pur
  • Majuscules on/off

Tout est sauvegarde dans localStorage et applique en temps reel. Echap ou clic sur la croix pour quitter.

📊 Statistiques d'ecoute

Onglet Stats, 4 sous-onglets :

  • Mes stats : total ecoutes, top heure, top artiste #1, plage active. Vues : Global / Annee / Mois / Jour avec drill-down. 5 designs : Classique / Vibrant / Ligne / Heatmap / Minimal
  • Ephemeres : titres ecoutes en streaming jamais sauves. Bouton ▶ orange pour re-ecouter directement, + Flow, DL, × Oublier
  • Stockage : choix de la strategie (mensuel / annuel / fichier unique)
  • Comparatif : evaluation ponderee des strategies selon des criteres cochables
👤 Profil & Preferences
  • Preferences par defaut : type prefere (audio/video), formats, qualites, couverture auto
  • Spectre visuel musical : effet d'arriere-plan anime pendant la lecture (opacite, flou, themes couleur, styles barres, sensibilite, etc.)
  • Onglet Recherche : nombre de recherches recentes affichees (10/20/50/100)
  • Lecture - Crossfade : duree du fondu enchaine (0-8s)
  • Mon Flow - Notifications & Paroles :
    • Notifier quand un titre demarre (necessite permission navigateur)
    • Ouvrir automatiquement le panneau Paroles a chaque nouveau titre
  • Cache : stats et bouton vider

💡 Astuces avancees
  • Workflow rapide : utilise l'extension Chrome v2 pour ajouter a Mon Flow en 1 clic depuis YouTube
  • Decouvertes mensuelles : la chip 📷 Decouvertes dans Mon Flow surface tes vrais coups de coeur recents (ajoutes <30j et ecoutes 3x+)
  • Comparer plusieurs versions de paroles : utilise ✏️ plusieurs fois, chaque recherche cree un onglet, switche entre eux
  • Recaler les paroles : si decalage, mets le mode click sur "Synchro" puis clique la ligne qui correspond a maintenant
  • Karaoke pour soiree : effet Bounce + couleur Or + police Display + background Aurora animee
  • Karaoke pour concentration : effet Statique + couleur Blanc + police Serif + background Noir pur
  • Transferer sur un autre PC : copie le dossier sans node_modules/, downloads/, data/, lance start.bat, puis Importer ton historique et Mon Flow
  • Sleep timer fade-out : 5/15/30/45/60/90 min ou custom, fade-out 5s a la fin
